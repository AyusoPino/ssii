# 💻 Sistemas Informáticos - DAW

Repositorio de trabajo del módulo **Sistemas Informáticos** del ciclo formativo de **Desarrollo de Aplicaciones Web (DAW)**.

En este repositorio se recopilan prácticas, ejercicios, scripts y recursos desarrollados durante el curso, con el objetivo de reforzar los conocimientos relacionados con la administración de sistemas, automatización de tareas y uso de herramientas fundamentales para el entorno profesional.

---

## 📚 Contenidos del módulo

Durante este módulo se trabajan, entre otros, los siguientes bloques:

- Arquitectura y componentes de un sistema informático
- Sistemas operativos
- Gestión de usuarios, permisos y recursos
- Línea de comandos
- Redes básicas
- Virtualización
- Seguridad informática
- Automatización con scripts
- Control de versiones con Git y GitHub
